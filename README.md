# Dirble

Gem to make interacting with http://api.dirble.com/ amazingly easy.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dirble'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dirble
